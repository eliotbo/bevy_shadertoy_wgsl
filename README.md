# bevy_shadertoy_wgsl

A [Shadertoy](https://www.shadertoy.com/) clone for the Bevy game engine, where the glsl language is replaced by wgsl.


Clone the repo, and for example run the following command:
```
cargo run --release --features bevy/dynamic --example paint_streams
```
